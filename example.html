Setup

This section covers how to set up to be able to use the Computer Vision APIs from Microsoft Cognitive Services.

Signing up to Microsoft

Microsoft offers various, readily available and powerful AI services, called Microsoft Cognitive Services . They provide various artificial intelligence APIs, with applications in areas such as speech, vision, language, knowledge and search. The web-based APIs make it possible for developers to easily utilize Microsoft's services via the web using HTTP requests to develop applications across various platforms. The APIs support many programming languages such as Python, C#, Java, etc. In this guide we will focus on using Javascript.
