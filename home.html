Introduction

This guide will go over how to use the Computer Vision API from Microsoft Cognitive Services to perform powerful computer vision tasks in your applications. Basic Javascript knowledge is required in order to follow the guide, but Microsoft Cognitive Services APIs support many different programming languages.

Microsoft Cognitive Services

Microsoft offers various, readily available and powerful AI services, called Microsoft Cognitive Services . They provide various artificial intelligence APIs, with applications in areas such as speech, vision, language, knowledge and search. The web-based APIs make it possible for developers to easily utilize Microsoft's services via the web using HTTP requests to develop applications across various platforms. The APIs support many programming languages such as Python, C#, Java, etc. In this guide we will focus on using Javascript.

Computer Vision API

Computer Vision API, is a more general computer vision service (in contrast to their other vision services such as Emotion, Face, or Content Modreation) that can be used to perform various, powerful computer vision tasks, such as image description, thumbnail generation, face recognition, tagging/categorizing images, estimating dominant colors, or capturing text from images. This guide cover how to use the "Analyze" API, which is a general, robust tool for image analysis. However, note that other Computer Vision APIs are called in much the same way, so learning how to use one will transfer to the other APIs (such as OCR (Optical Character Recognition), thumbnail generation, etc.).
